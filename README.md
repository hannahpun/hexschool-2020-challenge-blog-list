<h1 align="center">Hexschool 2020 鐵人賽文章搜尋器 v1.0.0</h1>

此服務目的在於搜尋 [六角學院 鼠年全馬鐵人挑戰](https://www.hexschool.com/2019/11/14/2019-11-14-w3Hexschool-2020-challenge/) 所參賽之文章，並基於六角學院釋出的 [API](https://github.com/hexschool/w3hexschool-API) 所開發的搜尋器。

服務連結：[Hexschool 2020 鐵人賽文章搜尋器](https://shawnlin0201.github.io/hexschool-2020-challenge-blog-list/)

## Features
- 關鍵字：搜尋文章標題（英文字母不分大小）
- 排序：更新日期遠近排序、發布文章數量多寡排序
- 其他：顯示參賽人次

許願功能以及狀況回報請到[這裡](https://github.com/shawnlin0201/hexschool-2020-challenge-blog-list/issues)回覆。
